## Conteúdo Abordado

A Avaliação Parcial 1 contempla tópicos introdutórios e fundamentais de Equações Diferenciais Ordinárias (EDOs), tais como:

- Equações diferenciais de primeira ordem
- Equações separáveis
- Equações lineares de primeira ordem
- Aplicações e interpretação das soluções
- Verificação de soluções

Todos os passos matemáticos estão descritos de forma detalhada no notebook.

## Como Executar o Notebook

1. Certifique-se de ter o Python 3 instalado.

2. Instale o Jupyter Notebook (caso ainda não tenha):

  ```bash
pip install notebook
```

4. Execute o notebook:

  ```bash
jupyter notebook AP1_2025_2.ipynb
```

## Exportação para HTML

O script `exportar.py` automatiza a conversão do notebook para HTML.

Para utilizá-lo, execute no terminal:
  ```bash
python exportar.py
```

O arquivo AP1_2025_2.html será gerado automaticamente na pasta do projeto.
